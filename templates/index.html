About EduGenAi

EduGenAi is a web-based platform that uses AI to generate quizzes from PDFs automatically. Teachers can save hours of preparation time, and students can access personalized quizzes for self-assessment. The system leverages Python, Flask, and PDF parsing for generating questions dynamically.

Key Features:

  • AI-powered quiz and answer generation
  • Multiple-choice question options
  • Downloadable quizzes for offline use
  • User-friendly interface with black-purple theme
  • Future-ready dashboard and analytics support
